🔐 Authentication & Security

Enterprise-grade security features

JWT Token-Based Authentication

Secure token-based authentication system with automatic expiration and refresh capabilities.

🔑 Token Generation

Automatic JWT token generation on successful login with 24-hour expiration

Authorization: Bearer {token}

🛡️ Password Security

Passwords are hashed using industry-standard algorithms

bcrypt with salt rounds

👤 User Registration

Self-service registration with email validation

POST /api/auth/register

Seamless shopping experience

Cart Operations

➕ Add to Cart

Add books with desired quantity to your cart

POST /api/cart/add

➖ Remove Items

Remove specific items from your cart

DELETE /api/cart/remove

🗑️ Clear Cart

Clear all items from your cart at once

DELETE /api/cart/clear

Real-Time Calculations

Cart totals are calculated in real-time based on current prices and quantities.

📦 Order Management

Complete order lifecycle

Order Workflow

  • Place Order: Convert cart items into an order
  • Order Confirmation: Immediate confirmation with order ID
  • Status Tracking: Real-time order status updates
  • Order History: View all past orders with details

Order Status

Orders progress through the following statuses:

  • Pending: Order received, awaiting processing
  • Processing: Order is being prepared
  • Shipped: Order has been dispatched
  • Delivered: Order successfully delivered
  • Cancelled: Order was cancelled

⚡ Performance & Optimization

Built for speed and scalability

Caching Strategy

Redis-based caching for frequently accessed data:

  • Book Catalog Cache: 5-minute TTL for book listings
  • Individual Book Cache: 10-minute TTL for book details
  • Order History Cache: 1-minute TTL for recent orders
  • Smart Invalidation: Cache automatically invalidated on updates

Database Optimization

  • Indexed queries for fast lookups
  • Pagination for large result sets
  • Covering indexes for common queries
  • Connection pooling

API Performance

  • Response compression
  • Async/await for non-blocking operations
  • Efficient serialization
  • Rate limiting support
